# Munchies Project

## Getting Started

To run the Munchies project, follow these steps:

1. **Navigate to Munchies Folder:**

   ```sh
   cd Munchies
   ```

```sh
npm install
```

```sh
npm run dev
```

2. **Navigate to sanity Folder:**

```sh
  cd sanity
```

```sh
npm install
```

```sh

```
